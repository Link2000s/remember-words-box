# 摸鱼克星：全网强制背单词解锁网站 🧠🔥

一个专治学习拖延症的硬核自律油猴脚本。当你忍不住想刷 B 站、逛微博时，必须先无条件背完规定数量的四级核心单词，网页才会对你解锁放行！

## 🚀 核心功能
* **强力拦截**：精准锁死 B 站、微博、知乎、抖音等主流摸鱼圣地。
* **科学记忆**：内置科学记忆曲线，拒绝无效死记硬背。
* **前后端解耦**：主逻辑与 2607 个核心词库分布式托管，轻量高效。
* **高度自定义**：支持自主修改每日词量，自由增减拦截黑名单。

## 🛠️ 用户高度自定义指南
* **修改每日词量**：打开主脚本代码第 14 行，将 `const DAILY_WORD_COUNT = 15;` 中的数字改为你想要的任意词量。
* **增加拦截网站**：在脚本顶部的 `// @match` 区域，照猫画虎添加你想戒断的网址即可（如贴吧、小红书）。
